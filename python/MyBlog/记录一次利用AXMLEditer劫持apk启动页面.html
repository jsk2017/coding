记录一次利用AXMLEditer劫持apk启动页面

2018年04月13日 01:54:26
阅读数：10

参考文章： http://www.wjdiankong.cn/archives/1036

前言

四哥真的很强！！学习记录了！

准备 

baksmali
smali
AXMLEditer

首先获取apk的dex文件，利用 baksmali 反编译成smali文件。
然后自己写个 demo ，同样反编译成 smali 。之后重新生成 classes.dex ，并利用压缩包重新打包。
接下来就需要使用 AXMLEditer 工具了，可以在 github 上查看源码和使用方法。不多介绍了。修改完 AndroidManifest.xml 文件后放回压缩包中，重新签名即可。注意一下，这里的操作要注意，尤其时往 AndroidManifest.xml 中插入标签时，注意语法的格式，否则在安装时很容易报错。

总结

虽然过程写的很简单，也没贴图，我这里只是做一个记录，具体的过程可以参考四哥的文章。我到最后虽然成功插入了，但是程序没有运行起来，停在微信的启动页面不动然后程序就死了，可能还是有点问题吧。
