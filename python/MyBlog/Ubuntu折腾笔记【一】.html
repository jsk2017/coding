前言

比较爱折腾吧，深度ubuntu用户。

开始

我的电脑是Thinkpad X1C 2nd
首先下载ubuntu的desktop镜像文件
使用 unetbootin-windows-638.exe 软件将镜像刻录到u盘上
U盘启动，安装Ubuntu即可。同安装windows类似。

更新

首先进入系统后，第一步要做的操作就是更新系统，系统也会提示你进行更新。 

sudo apt-get install update && upgrade

之后可以通过图形化的 ubuntu software 进行相应软件的安装
全新的ubuntu是没有net-tools工具包的，需要手动的安装 

sudo apt-get install net-tools

我觉得Ubuntu的界面也蛮漂亮的，有时间换一个小姐姐的桌面.嘿嘿嘿.
