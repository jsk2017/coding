can’t find lxxx解决办法

lxxx是libxxx的缩写。
首先我们可以在仓库中搜索一下有没有对应lib库 

apt-cache search libxxx

我就不截图了，反正我是知道了！
如果有对应的lib就使用apt-get install libxxx安装就行了
