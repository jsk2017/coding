前言

爱折腾！这两天可把我给折腾了一番。

软件

这里只列举一些比较使用的软件，具体的安装过程就不写了。
Docky 一个仿制mac osx的软件
Chrome 不用说了吧
guake 很方便的一款终端(透明的哦)
fcitx 搜狗拼音输入法，本来是想全英文工作的，但最后发现在搜索时需要用到中文啊，还有写博客的时候
albert 强大的启动器，比系统自带的搜索好用多了
Vim 神的编辑器，我感觉用习惯了是挺好用的，主要是方便
pycharm 具体的IDE看个人喜好了
tweaks gnome桌面美化的工具
synaptic 相当于是一个软件管家，可以卸载一些不知道怎么卸载的软件
nixnote2 据说是印象笔记的linux版本，但是为什么我的账号登不上去
gnome 可以通过访问他的官网下载很多好用的扩展和主题，用来美化界面还是不错的
kvm虚拟机本来想下来着，但是ubuntu好像不太好装，一般都是用centen os装的kvm
gparted 一款磁盘管理工具，比gnome-disks要好用我觉得是这样的
网易云音乐 linux版 这个好像需要从命令行启动才行，而且无法通过客户端进行注册401错误，但是有总比没有强，而且需要root启动，它的启动器设置好久没弄好。最后不得不通过终端起了。不过还是很美的
微信网页版 。。==.==
qq网页版没有找到 我记得多年以前还是有的呀
Todo 每天制定一个计划
sublime 蛮好用的，安装插件很方便
暂时就是这些，以后发现什么好用的软件在做补充！
————————-分割线——————————–
ps:感觉linux就是喜欢折腾界面，很漂亮，但我建议大家别入美化这个坑
创建软件的桌面快捷方式： 

vim /usr/share/applications/eclipse.desktop
[Desktop Entry]
Encoding=UTF-8
Name=eclipse
Comment=Eclipse IDE
Exec=/usr/local/eclipse/eclipse  #根据软件的具体执行路径修改
Icon=/usr/local/eclipse/icon.xpm  #根据软件的具体执行路径修改
Terminal=false  #软件打开时是否启动终端
StartupNotify=false
Type=Application
Categories=Application;Development;

总结

整体感觉thinkpad x1c安装ubuntu1804没有驱动的问题，小红点直接能用，无线，蓝牙驱动也都没有问题。看来ubuntu社区的工程师们做了很多的努力！辛苦了！
